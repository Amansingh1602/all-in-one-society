Backend (Node + Express + MongoDB)

- Copy `.env.example` to `.env` and set `MONGO_URI` and `JWT_SECRET`.
- Install dependencies: npm install
- Start dev server: npm run dev
- API endpoints:
  - POST /api/auth/register
  - POST /api/auth/login
  - GET /api/residents
  - GET /api/notices
  - POST /api/notices
  - GET /api/bookings
  - POST /api/bookings
